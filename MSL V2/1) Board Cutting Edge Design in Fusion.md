In Fusion 360 Sketch : 
- Go to File > Export > Save as DXF File (if you have Personal or Commercial License)
- If you have non commercial License : (for getting .dxf file )
	- After Completing the sketch, select the sketch and export the DXF
	- ![[Pasted image 20250920122000.png]]
	- 
	- Select the units and check mark sketch entities you want to export 
	- ![[Pasted image 20250920122059.png]]
	- Give the file name, select .dxf in file type and Choose the destination location and save
